MIMIC API

GET /proxies

List all known proxies

POST /proxies/register

Register a new proxy with all current and future brokers.

Params

proto (required)
The proxy's protocol (e.g. HTTP).
host (required)
The proxy's host.
port (required)
The proxy's port.
resp_time
The typical response time for this proxy.
geo
The geolocation for this proxy.
anon_level
The anonymity level (e.g. HTTP-ANONYMOUS.

POST /proxies/acquire

Acquire a proxy for a single request.

Params

url (required)
The URL of the page you want to request.
requirements
The comma-separated requirements for this request (e.g. HTTP-ANONYMOUS,US).
max_wait_time
The maximimum time to wait for a proxy resource before timing out.

POST /proxies/release

Acquire a proxy for a single request.

Params

broker (required)
The broker used when issuing a request from the returned proxy.
proxy (required)
The proxy to return.
response_time
The time it took from request initiation to completion.
is_failure
Set to `true` if this proxy did not work for the issued request.
  • POST /proxy/acquire?url=http://example.com/path&requirements=a,b,c
  • POST /proxy/release?proxy=http://proxyhost:port
  • GET /domains
  • GET /domains/{domain.com}
  • DELETE /domains/delete?domain=target.com

GET /domains

List the stats for all managed domains.

GET /domains/{domain}

List the stats for a specific domain.

DELETE /domains/{domain} (currently unimplemented)

Delete the monitoring of a domain.
